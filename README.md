# PDF Sentence Navigator

Chrome Extension (Manifest V3), რომელიც Chrome-ში გახსნილ PDF დოკუმენტში საშუალებას გაძლევს გადაადგილდე წინადადებებს შორის კლავიატურით.

## ფუნქციონალი

- Tab — გადასვლა მომდევნო წინადადებაზე
- Shift + Tab — დაბრუნება წინა წინადადებაზე
- აქტიური წინადადება ვიზუალურად ინიშნება ყვითელი ფონით
- მხარდაჭერილია მრავალგვერდიანი PDF დოკუმენტები
- მხარდაჭერილია როგორც ვებზე განთავსებული, ისე ლოკალური (file://) PDF-ები
- ყველა დამუშავება ხდება ლოკალურად ბრაუზერში — ტექსტი არასოდეს იგზავნება გარე სერვერზე

## ტექნოლოგიები

- JavaScript
- Chrome Extension Manifest V3
- PDF.js — ტექსტის ამოსაღებად და PDF-ის დასარენდერებლად (https://mozilla.github.io/pdf.js/)

## პროექტის სტრუქტურა

manifest.json — extension-ის კონფიგურაცია
background.js — service worker
popup.html / popup.js — extension-ის popup ფანჯარა
viewer.html / viewer.js — PDF viewer და navigation ლოგიკა
styles.css — viewer-ის და highlight-ის სტილები
icons/ — extension-ის icon-ები
pdfjs/ — PDF.js ბიბლიოთეკა (pdf.mjs, pdf.worker.mjs)

## ინსტალაცია (Load unpacked)

1. გახსენი Chrome-ში chrome://extensions
2. ჩართე Developer mode (გადამრთველი ზედა მარჯვენა კუთხეში)
3. დააჭირე Load unpacked
4. აირჩიე ეს პროექტის ფოლდერი (სადაც manifest.json მდებარეობს)
5. თუ გინდა ლოკალური (file://) PDF-ების გახსნაც:
   - extension-ის ბარათზე დააჭირე Details
   - ჩართე Allow access to file URLs

## გამოყენება

1. გახსენი PDF დოკუმენტი Chrome-ში (ვებზე ან ლოკალურად, file://)
2. დააჭირე extension-ის icon-ს browser-ის toolbar-ზე
3. Popup-ში დააჭირე Open PDF ღილაკს
4. ახალ ტაბზე გაიხსნება viewer, სადაც ტექსტი დამუშავებულია წინადადებებად და პირველი წინადადება ავტომატურად მონიშნულია
5. დააჭირე ერთხელ viewer-ის გვერდზე (ფოკუსის დასაყენებლად)
6. გამოიყენე:
   - Tab — შემდეგ წინადადებაზე გადასასვლელად
   - Shift + Tab — წინა წინადადებაზე დასაბრუნებლად

გვერდის ზედა ნაწილში (status ზოლში) ჩანს მიმდინარე პოზიცია, მაგალითად: წინადადება 5 / 42

## ფუნქციონალის შემოწმება (ტესტირების ინსტრუქცია)

1. აირჩიე ტექსტიანი PDF (არა სკანირებული სურათი — ტექსტი უნდა იყოს მონიშვნადი ჩვეულებრივ PDF viewer-შიც)
2. გახსენი extension-ის საშუალებით ზემოთ აღწერილი წესით
3. შეამოწმე:
   - პირველი წინადადება მონიშნულია ჩატვირთვისთანავე
   - Tab გადადის მომდევნო წინადადებაზე და ავტომატურად ასქროლავს, თუ საჭიროა
   - Shift + Tab ბრუნდება წინა წინადადებაზე
   - მრავალგვერდიან დოკუმენტში navigation გვერდებს შორისაც გამართულად მუშაობს
   - ბოლო წინადადებაზე Tab-ის დაჭერისას პოზიცია აღარ იცვლება (არ არსებობს "შემდეგი")
   - პირველ წინადადებაზე Shift+Tab-ის დაჭერისას პოზიცია აღარ იცვლება

## ცნობილი შეზღუდვები

- წინადადებების გამოცნობა ხდება პუნქტუაციის ნიშნებზე (. ! ?) დაფუძნებული რეგულარული გამოსახულებით — არასტანდარტული პუნქტუაცია ან შემოკლებები (მაგ. "და ა.შ.") შესაძლოა არასწორად გაიყოს ცალკე წინადადებებად
- სკანირებული (image-based) PDF-ები არ არის მხარდაჭერილი, რადგან მათში ამოსაღები ტექსტი არ არსებობს (საჭირო იქნებოდა OCR)
- დიდი მოცულობის PDF-ების შემთხვევაში ჩატვირთვას შესაძლოა გარკვეული დრო დასჭირდეს, რადგან ყველა გვერდი დამუშავდება ერთდროულად ჩატვირთვისას
